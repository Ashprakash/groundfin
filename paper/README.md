# Paper Workspace

This folder holds outlines, related work, experiment tables, figures, and draft material for the **Grounded Execution Precision (GEP)** papers.

## Paper 1: Method / Systems

Working title:

**Grounded Execution Precision: Auditable Extraction and Deterministic Execution for Financial Reasoning**

Main claim:

> Financial QA should not be framed only as retrieval-augmented generation. For numeric and computational financial questions, GEP constrains the language model to typed variable/formula extraction, performs computation in a deterministic executor, and reports committed-answer precision with explicit coverage.

## Paper 2: Benchmark / Diagnostic

Working title:

**GEP-Bench: Measuring Evidence-Grounded Execution Failures in Financial Decision Models**

Main claim:

> Existing financial QA evaluations hide the distinction between evidence access, variable extraction, computation, calibration, and abstention. GEP-style evaluation exposes which stage fails.
