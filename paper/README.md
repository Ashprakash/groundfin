# Paper Workspace

This folder will hold outlines, related work, experiment tables, figures, and draft material for the GROUNDFIN papers.

## Paper 1: Method

Working title:

**Grounded Probabilistic Distillation for Evidence-Conditioned Decision Models**

Main claim:

> Smaller student models can recover more teacher-level decision quality when supervision includes grounded evidence, uncertainty, calibration, and abstention signals rather than answer labels alone.

## Paper 2: Benchmark

Working title:

**FinGKD-Bench: A Counterfactual Benchmark for Grounded Financial Decision Reasoning**

Main claim:

> Existing financial QA benchmarks overestimate evidence-grounded reasoning because models can answer original examples while failing controlled counterfactual, missing-evidence, stale-evidence, and distractor-evidence variants.
